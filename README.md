# js-schema-6901

[JSON Pointer](http://www.rfc-base.org/txt/rfc-6901.txt) defines a string syntax for identifying a specific value within a JavaScript Object Notation (JSON) document.  For example, evaluating "foo/0/bar" against `{ foo: [ { bar: 1}, { bar: 2} }` returns 1;

[js-schema](https://www.npmjs.com/package/js-schema) is simple and intuitive object validation library.  It provides support for [JSON Schema](http://json-schema.org/).
